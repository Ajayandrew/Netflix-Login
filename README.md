# Netflix Login 🎉

A **React + Tailwind CSS** frontend project inspired by Netflix’s cinematic style.  
This dashboard celebrates **International Women’s Day** with a themed gallery, gradient background, and smooth UI.

---
## 🚀 Live Demo

🔗 Live Site: https://netflix-login-lilac.vercel.app/

## 📂 GitHub Repository

🔗 Repository: https://github.com/Ajayandrew/Netflix-Login.git

---

## 🚀 Features
- 🎨 Fixed red–black gradient background
- 📸 Women’s Day gallery with authentic images
- 🔐 Login & Logout flow (connected to backend API)
- 📱 Responsive design with Tailwind CSS
- 🎬 Netflix‑style dashboard layout
